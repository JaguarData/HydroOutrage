To use this extension. 

Download the code to a new folder and put it somewhere on your computer

Open a new chrome tab.

go to chrome://extensions/

Click on the "Load Unpacked" 

If that's not available maybe try to Toggle on Developer mode if not enabled. Should be  switch in top Right.

Select the folder  you downloaded the file to.

Go to the following URL to test it out

https://hydroottawa.com/en/outages-safety/outage-centre
